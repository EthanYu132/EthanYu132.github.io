Notification System

Python, Selenium, Firebase

As a Computing consultant for Rice University, part of me and my co-workers jobs are to answer the numerous emails we get every day requesting help regarding technology. Because Rice uses their own proprietary system to handle these emails, their notification system is not robust whatsoever, leading to slow response times in said emails. I used Python's Selenium library and Firebase to scrape data and programmatically send email notifications to each one of my co-workers ensuring that each help request is answer promptly, decreasing our average response time from approximately 2-3 hours per request to under an hour per request.
